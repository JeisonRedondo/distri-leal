# Instrucciones para el asistente

## Idioma
Responde SIEMPRE en español. Sin excepciones, incluso para comentarios de código.

## Proyecto
- Nombre: distri-leal
- Stack: React + Vite, JSX, CSS modular por componente
- Ruta: ~/proyectos/distri-leal

## Estructura
src/
├── App.jsx              # Componente raíz, enrutamiento principal
├── main.jsx             # Entry point
├── data/products.js     # Datos de productos
├── styles/globals.css   # Estilos globales
└── layouts/
    ├── Header/          # Navegación principal
    ├── Footer/          # Pie de página
    ├── MainLayout/      # Layout base
    ├── CategoriesSection/  # Sección de categorías
    └── FeaturedProducts/   # Productos destacados

## Al iniciar sesión
Lee estos archivos automáticamente para entender el proyecto:
- src/App.jsx
- src/main.jsx
- src/data/products.js
- src/styles/globals.css

## Convenciones
- Componentes en PascalCase
- Cada componente tiene su propio CSS en la misma carpeta
- Los layouts exportan desde index.js cuando existe
- Comentarios en español

## Lo que NO debes hacer
- No reescribir archivos completos si solo se pide un cambio pequeño
- No agregar dependencias sin preguntar primero
- No modificar globals.css ni index.css sin que se pida explícitamente
- No cambiar la estructura de carpetas existente
